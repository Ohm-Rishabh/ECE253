# ECE253

This is the repo for the course project ECE253 at UCSD

## Requirements
- **MATLAB** (recommended: R2020a or newer)
- **Toolboxes**
  - Image Processing Toolbox
  - Statistics and Machine Learning Toolbox
  - https://webpages.tuni.fi/foi/GCF-BM3D/index.html 

## Get ISETCam
Clone the ISETCam simulator:
```bash
git clone https://github.com/ISET/isetcam.git
```

Run Code:
```bash
run("file_name.m")
```
